INSTRUCCIONES PARA INICIAR EL FRONTEND (React + Vite)

1. Instalar Node.js y npm (si no lo tienes instalado)
   - Descarga desde https://nodejs.org/

2. Crear el proyecto React con Vite (solo si no existe la carpeta 'frontend'):
   npm create vite@latest frontend -- --template react

3. Entrar a la carpeta del frontend:
   cd frontend

4. Instalar las dependencias:
   npm install

5. Ejecutar el servidor de desarrollo:
   npm run dev

6. Acceder a la aplicación en el navegador:
   - Normalmente en http://localhost:5173

Notas:
- Si ya existe la carpeta 'frontend', solo realiza los pasos 4 y 5.
- Puedes instalar librerías adicionales con 'npm install nombre_paquete'.
- Para detener el servidor, presiona Ctrl+C en la terminal.
